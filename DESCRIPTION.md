Repo description (short, for GitHub "About" field)

Signal Desk — a shared, live opportunity tracker. Log, score, and rank business/product ideas across domains (AI, healthcare, finance, climate, and more) with an 8-axis weighted scoring model. Single static HTML page, backed by Supabase, no login — anyone with the link can add or edit entries.

A single-page opportunity engine for logging and ranking startup/product ideas across domains, backed by a shared Supabase database.
